# petmo-design-system

custom design system for petmo site

## Documentation

### Installation

```
npm install petmo-design-system
```

### Usage

- You can use theme and globalstyle at once
```jsx
const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(
  <PetmoStyleProvider>
    <App />
  </PetmoStyleProvider>
);
```
